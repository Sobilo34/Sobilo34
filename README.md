## About Me

I'm a **Full-Stack Engineer** with roots in both Web2 and Web3. My journey started at **Elite Computer Technology** learning frontend development and building web solutions. **ALX Software Engineering** sharpened me with systems programming, algorithms, and industrial-level projects. **Web3bridge** cemented my blockchain engineering foundation — smart contracts, dApps, DeFi, and beyond.

I build things that matter. Whether it's a clean REST API, a gas-optimized smart contract, or a responsive web or mobile app connected to the blockchain — I care about craft, correctness, and impact.

When I'm not shipping code, I'm contributing to open-source, partnering with early-stage startups, and exploring where AI meets decentralized systems.

> *"The best engineers aren't just those who write code — they're those who understand why."*
